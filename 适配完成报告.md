## ⚠️ 注意事项

### 1. 权限要求
某些功能需要 root 权限：
- WiFi 相关功能
- DOS 攻击（使用原始套接字）

使用方法：`sudo python3 ANSportion.py`

### 2. WiFi 功能限制
Mac 的 WiFi API 有限制：
- pywifi 库功能受限
- 需要使用系统命令 (networksetup)
- 需要管理员权限

### 3. 防火墙设置
确保防火墙允许 Python 的网络访问：
- 系统偏好设置 → 安全性与隐私 → 防火墙

### 4. 法律警告
